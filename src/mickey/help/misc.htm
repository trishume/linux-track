Miscelaneous setup

Hotkey setup

Hotkey setup allows you to customize the hotkey for starting/stopping the tracking. It covers some seldom used keys, that should be suitable for the purpose.

The hotkey allows you to select the hotkey and possibly a modifier (Ctrl, Alt, ...). If the hotkey is not usable (e.g. it is already grabbed by another application), a warning messagebox pops up and the previous hotkey is reset.

Timeouts setup

To facilitate a hands free setup of Mickey, all the settings are timed - there is no need to click any buttons.

Centering timeout specifies a time after the press of the Recenter button and the actual center point registration.

Calibration timeout is a time during which you are supposed to move your head to left/right/top/bottom extremes (that are still usable for the tracking) during calibration. When the time is up, the calibration is finished and the tracking is adjusted appropriately.
