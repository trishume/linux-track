Miscelaneous

The last pane contains things that didn't fit into the other panes.

Xplane plugin installation

This button installs the XPlane plugin. When pressed, a dialog appears asking you to browse to your XPlane executable. When there, select the executable and press button Open. In case of problem, a message is shown, otherwise you can close ltr_gui and start XPlane.

In case you use several XPlane installations, install the plugin into each of them.

For details on the XPlane plugin setup see Xplane plugin setup page.

Wine plugin installation

This button lets you install the linuxtrack-wine bridge. It allows TrackIR enabled games in wine to utilize linuxtrack.

When you press the button, you'll be asked to go to the "wine prefix" (or bottle on Mac) of your choice. If you are unsure what I'm talking about, just press enter and the default one is used. An installator shows up - follow the instructions and when done, start the Controller.

Controller window shows and you should specify pause key and recenter key - just press the appropriate button followed by key/keys of your choice (max 2 keys). These keys will be used ingame to pause the tracker (e.g. when playing with avionics).

When the controller is running, start the program/game and the tracking should just work!

When using more prefixes, always make sure to install the plugin into all of them and also make sure you use the Controller from the same prefix, otherwise you'll not be able to controll the linuxtrack!

Logfile viewer

Press the button to see the ltr_gui log - can contain some interesting informations when debuggin a problem.

Package Logs

This button creates zip file containing Linuxtrack logs, which is handy when reporting a problem.

Troubleshooting options

There are several checkboxes allowing you to modify Linuxtrack behavior when needed...
  • Disable camera view disables camera view in the Tracking window.
  • Disable 3D tracking preview disables 3D preview in the Tracking window; this might be handy especially on machines with weak GPU (gma950 and similar), where it considerably lowers the machine load.
  • Use legacy pose computation method - not to be checked unless directly asked by developer.
  • Use legacy rotation computation method - not to be checked unless directly asked by developer.
  • Disable translation/rotation alignment allows you to decouple rotations and translations. Normally translations are aligned with in-game view - when you rotate your in-game head to look out of the side window and move towards the screen, the view moves towards the window. If this behavior is not desired, check the button and the effect will be disabled.
