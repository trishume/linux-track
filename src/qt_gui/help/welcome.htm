Welcome to Linuxtrack!

First of all, thank you for using Linuxtrack, headtracking software for Linux and MacOS X.

Linuxtrack allows you to use a webcamera, wiimote or any of the supported NP devices (currently TrackIR2 to 5, SmartNav3 and SmartNav4) as a headtracking device, facilitating view control in simulators (flight, racing) or any other type of software.

Among applications utilizing Linuxtrack are for example XPlane flight simulator, FlightGear flight simulator, and any TrackIR enabled game running in Wine (Condor soaring simulator, Il-2, Falcon 4, ...).

If you want to know more, see the following pages:

  • Linuxtrack Project page
  • XPlane.org Forums, namely its Linux forum
  • Freetrack project page - the "model" creation hints, tutorials and more
Should you encounter a bug, please take a few minutes and file a bug report at the project page - it will allow me to fix the bug that is bothering you, and it also helps to improve Linuxtrack itself. Also any comments, ideas for improvement and constructive criticism is very welcome!
