Linuxtrack - Wine

To allow Windows programs running under Wine to use headtracking, Linuxtrack provides Linuxtrac-Wine installer. This installer provides headtracking both to TrackIR enabled applications (both native like Condor Soaring, or via TIRViews.dll like MSFS series) and also FreeTrack compatible applications.

To enable headtracking, just install the linuxtrack-wine.exe to the bottle/prefix of your choice either from ltr_gui, or directly.

Please note, that it is necessary to install TrackIr firmware (along with mfc42u.dll needed by TIRViews.dll) to utilize all of the above options. The easiest way to do that is to use the Install Linuxtrack-Wine support... button. That way, all the necessary components will be installed/extracted in one go.

If you installed Linuxtrack-Wine prior to version 0.99.11, to gain access to the new features you need to uninstall it first, reinstall TrackIR firware (to get TIRViews.dll and mfc42u.dll) and then install the new linuxtrack-wine.exe.
